The NVIK circos figures is generated by "circos | v 0.69 | 6 Dec 2015 | Perl 5.018002" using these files.
Please download, extract, and install circos-0.69.tgz from http://circos.ca/software/download/circos/

copy the folder NVIK circos in the circos directory and run the following command to generate the circos diagram

Step 1: Go to NVIK_circos folder
run the command
../bin/circos -conf circos1.conf

this will reproduce two image files circos.svg and circos.png.
